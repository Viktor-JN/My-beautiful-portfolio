HTML

STRUKTUR

<!DOCTYPE>

Definierar dokumenttypen och versionen av HTML-dokumentet.

<Html>

Huvudelementet som omger hela HTML-dokumentet.

<Head>

Innehåller metadata om dokumentet, såsom titel, länkar till externa resurser och stilinformation.

<Title>

Används för att ange titeln på dokumentet, vilket visas i webbläsarens flik eller fönster.

<Style>

Används för att inbädda interna stilmallar (CSS) direkt i dokumentet.

<Main>

Används för att definiera huvudinnehållet på en webbsida.

SEMANTISKA ELEMENT

<Header>

Används för att definiera sidans huvudsektion, vanligtvis innehållande rubriker, logotyper och navigationsmenyer.

<Link>

Används för att länka till externa stilmallar (CSS) eller andra resurser.

<Body>

Innehåller det synliga innehållet på webbsidan, såsom text, bilder och andra element.

<p>

Skapar en styckeavsnittstext.

headings>

(<h1-<h6) Skapar rubriker med olika hierarkiska nivåer, där h1 är den högsta nivån och h6 är den lägsta.

<ol>

Skapar en ordnad lista, där listobjekten numreras.

<ul>

Skapar en oordnad lista, där listobjekten inte har någon särskild ordning.

<li>

Används inuti listelementen (ol eller ul) för att skapa individuella listobjekt.

<strong>

Används för att markera text som bör vara extra betonad och oftast visas fett.

<em>

Används för att betona texten och oftast visas kursivt.

<a>

Skapar en länk till en annan webbsida eller resurs.

<span>

Ett generiskt in-line element som används för att applicera stilar eller manipulation på en del av en text eller innehåll.

<section>

Används för att gruppera innehåll i ett tematiskt block inom dokumentet.

<footer>

Används för att definiera sidans fotsektion, vanligtvis innehållande upphovsrättsinformation och kontaktuppgifter.

<article>

Används för att definiera ett oberoende och självständigt innehållsstycke, t.ex. en bloggpost eller en nyhetsartikel.

<aside>

Används för att definiera sidorutor eller innehåll som är sidbetecknat och inte nödvändigtvis en del av huvudinnehållet.

<div>

Ett generiskt blockelement som används för att gruppera och strukturera innehåll och tillämpa stilar.
